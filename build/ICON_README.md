# アプリアイコンについて

インストーラーにアイコンを含めるには、以下のファイルをこのディレクトリに配置してください：

## Windows用
- `icon.ico` - 256x256ピクセル以上のICOファイル

## macOS用（オプション）
- `icon.icns` - 512x512ピクセル以上のICNSファイル

## Linux用（オプション）
- `icon.png` - 512x512ピクセル以上のPNGファイル

## アイコンファイルがない場合

アイコンファイルがなくても、アプリケーションは正常にビルドされます。
デフォルトのElectronアイコンが使用されます。

## アイコンの作成方法

1. **オンラインツールを使用**
   - https://icon.kitchen/ - 無料のアイコン生成ツール
   - https://www.icoconverter.com/ - PNG→ICO変換

2. **デザインツールを使用**
   - Figma, Canva, Adobe Illustratorなど
   - 推奨サイズ: 512x512ピクセル

3. **シンプルなデザイン例**
   - PDFを表す書類アイコン
   - 編集を表すペンやツールアイコン
   - 企業/プロジェクトのロゴ
